# Tech Stack Decision

Решение принято:

1. **Язык:** Python (без JS/TS)
2. **Типизация:** строгая (mypy strict)
3. **Модели:** Pydantic v2
4. **Архитектура:** Clean Architecture + DDD
5. **Внутренняя шина:** EventBus (in-memory на MVP)

## Правила кода
- Любая функция должна иметь полные type hints
- `Any` запрещён, кроме явно обоснованных случаев
- Бизнес-правила — только в domain/application
- Инфраструктура подключается через интерфейсы
